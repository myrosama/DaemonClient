## 🚀 Deployment Instructions

### Prerequisites
1. Cloudflare account with API token
2. Firebase project with Firestore
3. Wrangler CLI installed (`npm install -g wrangler`)

### Step 1: Deploy Deployment Service
```bash
cd deployment-service
wrangler secret put ENCRYPTION_MASTER_KEY
# Enter a strong 32-character key
wrangler deploy
# Note the worker URL for next step
```

### Step 2: Update Accounts Portal Environment
```bash
cd accounts-portal
echo "VITE_DEPLOYMENT_SERVICE_URL=https://daemonclient-deployment.YOUR_ACCOUNT.workers.dev" >> .env
npm run build
firebase deploy --only hosting
```

### Step 3: Test End-to-End
1. Visit accounts.daemonclient.uz
2. Create test account
3. Complete Telegram setup
4. Complete Cloudflare setup
5. Verify worker deployed
6. Upload photo and verify it works
